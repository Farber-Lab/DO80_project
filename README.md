# DO80 Project

Code accompanying the publication:

**Dillard LJ, Calabrese GM, Mesner LD, Farber CR (2024)**  
Cell type-specific network analysis in Diversity Outbred mice identifies genes potentially responsible for human bone mineral density GWAS associations.  
*eLife* 13: RP100832.

https://doi.org/10.7554/eLife.100832.2

## Contents

- `network_paper_code.R` – R code used for network analysis in the DO80 project.

## Description

This repository contains analysis code used to perform cell type–specific network analysis in Diversity Outbred mice to identify genes potentially responsible for human bone mineral density (BMD) GWAS associations.
